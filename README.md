# Server monit Telegram Bot for Alexx6
## Command run application
To build the application, run this line in the console:
### Build
```
 npm run App:build
```

### Start
```
 npm run App:start
```

*You may need to install the **npm**: version - 9.5.0 or latest, **node**: version - v18.15.0  or latest.*
